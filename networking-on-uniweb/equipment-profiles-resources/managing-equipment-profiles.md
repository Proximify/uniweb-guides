# Gestion d'équipements

Les équipements **ne sont pas** spécifiquement liées aux rôles d'administrateur, et tout membre d'UNIWeb peut donc répertorier ses propres équipements, qu'il dispose ou non d'autorisations d'administrateur. Certaines équipements peuvent être gérées par plusieurs personnes au fil du temps, et vous pouvez donc transférer la propriété de votre équipement à un collègue afin qu'il puisse gérer l'équipement et répondre aux demandes à l'avenir.

Une fois qu'un visiteur d'UNIWeb vous a contacté afin d'utiliser votre équipement, toute autre communication se fera en dehors d'UNIWeb; UNIWeb ne propose aucun moyen de négocier des transactions pour des biens ou des services offerts via les équipements.

{% hint style="danger" %}
**À noter:** Les administrateurs auxquels un [rôle d'administrateur](../../uniweb-accounts/access-control/managing-administrator-roles-and-permissions.md) a été attribué qui inclut l'autorisation Modifier les profils d'équipement peuvent effectuer les procédures ci-dessous pour tous les équipement sur leur réseau UNIWeb, qu'ils soient le propriétaire de l'équipement ou que le propriétaire se trouve dans l'unité académique de l'administrateur.
{% endhint %}

## **Création d'un équipement**

1. À partir de votre page [d'accueil UNIWeb](../../navigating-uniweb/the-home-page.md), allez dans équipements**.**
2. Dans le panneau central, cliquez sur **Ajouter un équipement.**
3. Dans la boîte de dialogue Ajouter un équipement, entrez les [**information générales**](managing-equipment-profiles.md#resource-general-information) de l'équipement dans le formulaire de saisie de données. Les champs marqués d'un astérisque \(\*\) sont obligatoires.
4. Cliquez sur **Enregistrer.**

## **Modification d'un** équipement

1. À partir de votre page [d'accueil UNIWeb](../../navigating-uniweb/the-home-page.md), allez dans **équipements**.
2. Dans le panneau central, recherchez et cliquez sur l'équipement que vous souhaitez modifier. Cela vous amènera au profil public de cet équipement.
3. Dans le panneau de droite, cliquez sur **Modifier** le profil. Cela vous amènera à l'éditeur de profil pour cet équipement.
4. Dans le panneau central, accédez à la [section profil](managing-equipment-profiles.md#resource-profile-sections) que vous souhaitez modifier. Cliquez sur le bouton Modifier à droite du titre de la section. Vous pouvez modifier les [informations générales](managing-equipment-profiles.md#resource-general-information) de l'équipement en cliquant sur le bouton Modifier à droite du nom de l'équipement.
5. Ajoutez ou modifiez des informations dans le formulaire de saisie de données pour la section de profil de l'équipement correspondant, si nécessaire.
6. Cliquez sur **Enregistrer.**

## **Suppression d'un** équipement

1. À partir de votre page [d'accueil UNIWeb](../../navigating-uniweb/the-home-page.md), allez dans **équipements**.
2. Dans le panneau central, **recherchez et cliquez sur l'équipement** que vous souhaitez supprimer. Cela vous amènera au profil public de cet équipement.
3. Dans le panneau de droite, cliquez sur **Modifier le profil.** Cela vous amènera à l'éditeur de profil pour cet équipement.
4. Dans le panneau de droite, cliquez sur **Supprimer le profil.**
5. Dans la boîte de dialogue **Supprimer le profil,** cliquez sur le bouton **Oui** pour confirmer que vous souhaitez supprimer l'équipement sélectionné.

## **Transfert de propriété d'un** équipement ****

1. À partir de votre page [d'accueil UNIWeb](../../navigating-uniweb/the-home-page.md), allez dans **équipements**.
2. Dans le panneau central, recherchez et **cliquez sur l'équipement** que vous souhaitez transférer à un nouveau propriétaire. Cela vous amènera au profil public de cet équipement.
3. Dans le panneau de droite, cliquez sur **Modifier le profil.** Cela vous amènera à l'éditeur de profil pour cet équipement.
4. Dans le panneau de droite, cliquez sur **Transférer la propriété du profil.**
5. Dans la boîte de dialogue Transférer la propriété du profil, sélectionnez l'utilisateur UNIWeb auquel vous souhaitez transférer la propriété de l'équipement.
6. Cliquez sur envoyer.

Le nouveau propriétaire potentiel recevra une notification vous informant que vous lui avez demandé de devenir propriétaire de l'équipement. S'ils acceptent, UNIWeb leur transférera la propriété de l'équipement.

{% hint style="danger" %}
**À noter:** si vous ne disposez pas d'un [rôle ](../../uniweb-accounts/access-control/managing-administrator-roles-and-permissions.md)[d'administrateur](../../uniweb-accounts/access-control/managing-administrator-roles-and-permissions.md) qui inclut l'autorisation de modifier les profils des équipements, vous perdrez toute possibilité de modifier le profil de l'équipement une fois que le nouveau propriétaire aura accepté le transfert.
{% endhint %}

## **Création d'une sauvegarde d'un** équipement ****

1. À partir de votre page [d'accueil UNIWeb](../../navigating-uniweb/the-home-page.md), allez dans **Équipements**.
2. Dans le panneau central, recherchez et cliquez sur l'équipement dont vous souhaitez créer une sauvegarde. Cela vous amènera au profil public de cet équipement.
3. Dans le panneau de droite, cliquez sur **Modifier le profil.** Cela vous amènera à l'éditeur de profil pour cet équipement.
4. Dans le panneau de droite, cliquez sur **Créer une copie de sauvegarde.**

UNIWeb téléchargera une copie de sauvegarde de l'équipement sur votre ordinateur sous forme de fichier JSON.

</table>### **Sections de profil de l'**équipement

Vous pouvez utiliser les sections de la page de profil d'un équipement pour donner des informations plus approfondies concernant l'équipement elle-même, la façon dont elle est censée être utilisée et les conditions dans lesquelles elle est disponible. Si vous laissez une section vide, elle ne sera pas visible par les visiteurs sur la page de profil de l'équipement

| **Section Profil** | **Objectif** |
| :--- | :--- |
| **Description** | Fournissez une description générale de l'équipement, ainsi que toute information permettant aux visiteurs de mieux comprendre son objectif. |
| **Caractéristiques** | Décrivez tous ls aspects techniques de l'équipement si nécessaire. Vous pouvez joindre des fichiers à télécharger par les visiteurs pour plus d'informations. |
| **Conditions d'utilisation** | Indiquez si l'équipement est **disponible par collaboration**, ou est disponible moyennant des **frais de service.** Vous pouvez indiquer les tarifs des services, le cas échéant, et vous pouvez joindre des fichiers ou des formulaires que les visiteurs peuvent télécharger pour plus d'informations. |
| **Emplacements** | Énumérez les emplacements où l'équipement est disponible. Les emplacements seront ajoutés à l'écran Carte. |
| **Disponibilité** | Énumérez les jours de la semaine et les heures de ces jours où l'équipement est disponible pour utilisation. Vous pouvez également indiquer s'il y a un opérateur sur place pour aider à l'utilisation de la équipement pendant ces périodes. |
| **Thèmes de recherche** | Marquez votre équipement avec des [Thèmes de recherche](../research-themes/increasing-discoverability-with-research-themes.md#tagging-resources-with-research-themes) ****pour augmenter le nombre de mots clés que les visiteurs peuvent utiliser pour trouver votre équipement dans les résultats de recherche ou en filtrant sur la page Équipement. Votre équipement sera également affichée dans le cluster de recherche correspondant au thème de recherche étiqueté. |
| **Secteurs d'application** | Dans une liste déroulante de sélections disponibles, sélectionnez un secteur qui décrit le mieux le domaine de recherche pour lequel l'équipement est conçue. |
| **Information sur les installations de recherche** | Ajoutez le nom et l'URL du centre de recherche qui héberge ou gère l'équipement. |
| **Informations de contact alternatives** | Ajoutez toute autre adresse e-mail ou numéro de téléphone pouvant être utilisé pour vous renseigner sur l'utilisation de l'équipement. |
| **Membres** | Gérez les membres d'UNIWeb qui peuvent trouver et consulter cet équipement au sein du réseau UNIWeb de votre institution. Cette section n'est disponible que si vous avez choisi l'option **Membres**, sous **Visibilité** dans les informations générales de l'équipement. |
| **Mots clés** | Ajoutez des mots clés qui se rapportent à l'équipement ou à son sujet. Cela augmentera sa découvrabilité dans les résultats de recherche. |
